# CLYK Copywriting Skill

A Claude skill that gives you Harry Dry, Eugene Schwartz, and Michael Masterson 
in your computer — automatically applied to every ad, hook, script, and landing 
page you write.

## What it does
- Runs Harry Dry's 3-rule test (visualize, falsify, nobody else) on every headline
- Builds copy from Eugene Schwartz's claim + mechanism architecture
- Enforces Masterson's Rule of One — one big idea, no toss salad
- Generates hooks (6 types), video scripts, visual concepts, and UGC briefs
- Audits existing copy and rewrites weak elements

## What's inside
- Full copywriting workflow (brief → vocabulary → write → audit)
- Annotated breakdowns of 9 famous ads
- Claim exhaustion + twist system for fatigued offers
- Hook-to-script pipeline for Meta, TikTok, and Reels
- UGC talent brief template
- Quick-start prompts for every common copy task

## How to install
1. Download `clyk-copywriting.skill`
2. Go to Claude.ai → Settings → Skills → Install skill
3. Select the downloaded file
4. Done — activate it in any conversation
